1. 환경규제 확인 프로세스 (업무 흐름도)

프로세스 단계 설명:
1. 신규 원재료 도입 / 정기 점검 → 원재료 SDS(안전보건자료) 확보
2. CAS No. 추출 및 시스템 등록
3. 5종 규제 DB 교차 확인: EU REACH SVHC(512종), EU RoHS(10종), PFAS(877종), US TSCA(Active/Inactive), CA Prop65(819종)
4. 규제 해당 여부 판정 → 미해당 시 적합(PASS)
5. 해당 시 리스크 레벨 평가 → HIGH(2종 이상) / MEDIUM(1종)
6. HIGH: 대체물질 검토 및 함량 저감 / MEDIUM: 모니터링 및 관리 계획
7. 제품별 규제 영향 평가 → 거래처 통보 및 규제 적합 증명서 발행 → 정기 업데이트 반영

2. 시스템 데이터 구조 (ER Diagram)

테이블 설명: PRODUCT(제품코드, 배합코드, 납품명, 거래처) | FORMULATION(배합코드, 제품코드) | RAW_MATERIAL(원재료코드, 원재료명, 공급처, 제조사, 사용여부) | CAS_COMPONENT(CAS번호, 국문명, 영문명, 함유량, 유독물질번호, 제한물질번호) | REGULATION_CHECK(CAS번호, REACH, RoHS, PFAS, TSCA, CA65, 리스크레벨) | EU_REACH_DB(물질명, EC번호, CAS번호, 포함사유, 등재일) | EU_ROHS_DB(물질명, CAS번호, 제한기준) | PFAS_DB(물질명, CAS번호) | US_TSCA_DB(ID, CAS번호, 화학물질명, Active/Inactive) | CA65_DB(물질명, CAS번호, 독성유형)

환경규제 전문가 관점 - 고려사항 분석

A. 즉시 조치 필요 사항 (Critical)

1. DEHP (117-81-7) - 3중 규제 해당 물질

현황: EU REACH SVHC + EU RoHS + CA Prop65 동시 해당. 원재료코드 OD001A/B에서 사용 중(Y).
리스크: EU 수출 시 SCIP DB 신고 의무, RoHS 제한기준(0.1%) 초과 시 판매금지, 캘리포니아 판매 시 경고 라벨 의무.
조치: 제품 내 DEHP 함유량 정밀 분석 → 제한기준 이내 확인 → 대체 가소제(DINP, DOTP 등) 전환 검토.

2. DecaBDE (1163-19-5) - EU REACH + RoHS 이중 규제

현황: 원재료코드 FH009A에서 함유량 100%로 사용 중. PBDE 계열 난연제.
리스크: EU POPs 규정에도 해당 가능. 2021년부터 EU 내 사실상 사용 금지.
조치: 즉시 대체 난연제(Al(OH)3, Mg(OH)2, 인계 난연제) 전환 필요. 재고 소진 후 사용 중단.

3. n-Hexane (110-54-3) - 2026년 2월 신규 SVHC 추가

현황: 최근 SVHC 후보목록 추가(2026.02.04). 원재료코드 EE011H, EE043A/Z, EE044A에서 사용 중.
리스크: 반복노출 표적장기 독성(Art.57f). 향후 인허가 대상 전환 시 사용 제한 가능성 높음.
조치: 대체 용매 검토(이소헥산, 헵탄 등). 공정 내 밀폐/환기 강화. SCIP DB 신고 준비.

B. 모니터링 및 관리 필요 사항 (Warning)

4. ETU - 에틸렌티오우레아 (96-45-7)

현황: EU REACH SVHC + CA Prop65 해당. 고무 가황촉진제로 사용(CA016A/B).
리스크: 생식독성 물질. EU 인허가 대상 전환 가능성.
조치: 대체 촉진제(DPTU, DBTU) 검토. 작업환경 노출 모니터링 강화.

5. ADCA - 아조디카본아미드 (123-77-3)

현황: EU REACH SVHC 해당. 발포제로 다수 원재료에서 사용(CI008A/B, CI012A, CI014A, CI015A).
리스크: EU 식품접촉재료에서 이미 사용 금지. 산업용도 제한 확대 가능성.
조치: 대체 발포제(마이크로스피어, 소듐비카보네이트) 적용 테스트. 용도별 규제 적용 범위 확인.

6. 4-tert-Octylphenol (140-66-9) / Nonylphenol (84852-15-3)

현황: 알킬페놀류 REACH SVHC 해당. 접착제/코팅제 원료로 사용 중.
리스크: 내분비 교란물질. EU 제한물질 등재 추진 중.
조치: 알킬페놀 프리 대체제 확보. 공급업체 대체품 공급 가능 여부 확인.

7. CA Prop 65 단독 해당 물질 (클로로프렌, 톨루엔, MBT 등)

현황: 미국 캘리포니아 수출 시 경고문 부착 의무.
리스크: 경고 라벨 미부착 시 소송 리스크 (바운티헌터 소송).
조치: 캘리포니아 판매 제품 식별 → 경고문 부착 여부 확인 → Safe Harbor Level 확인.

C. 시스템 개선 권고사항

8. 규제 DB 자동 업데이트 체계 구축

현황: Excel 기반 수동 관리. ECHA/EPA 사이트 수동 확인.
권고: ECHA REST API, EPA CompTox Dashboard 연동으로 자동 업데이트. SVHC 추가/변경 시 자동 알림 시스템.
효과: 규제 변경 누락 방지, 대응 리드타임 확보 (현재 수개월 → 실시간).

9. 제품-원재료-CAS 연계 데이터 정비

현황: 배합원재료정리 시트에서 제품→원재료 매핑이 불완전 (Col 4~24 빈 영역).
권고: ERP 배합(BOM) 데이터와 연계하여 제품별 포함 원재료 자동 매핑. CAS 성분까지 추적 가능한 체계 구축.
효과: 제품 단위 규제 영향 즉시 파악, 거래처 문의 시 신속 대응.

10. PFAS 규제 선제 대응

현황: 현재 PFAS 해당 물질 0건이나, EU에서 전면 금지 추진 중 (2026년 상반기 심의).
권고: 사용 원재료 내 불소 함유 화합물 전수 조사. 공급업체 PFAS-Free 선언서 확보.
효과: 규제 시행 시 즉시 적합 증빙 가능. 선제적 대체물질 확보로 공급 리스크 최소화.

11. 거래처별 규제 대응 커뮤니케이션

현황: 디엔테크(EPS 시리즈), 동남무역(TE/TR/R 시리즈) 2개 거래처 대상 17개 제품 납품.
권고: 거래처별 수출 대상국 확인 → 해당국 규제 기준 적용. 규제 적합 증명서(Declaration of Conformity) 정기 발행. SCIP 번호 제품별 부여 및 공유.
효과: 거래처 신뢰 확보, 수출 통관 지연 방지.
